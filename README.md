## Installing

Install requirements:

* python
* tox

```console
tox
```

To reinstall the python virtual environment:

```console
tox -r
```

## Activating the `tox` environment

```console
source .tox/python/bin/activate
```

## Running the server

``console
esphome dashboard config
```

The URL is [http://127.0.0.1:6052/](http://127.0.0.1:6052/).
